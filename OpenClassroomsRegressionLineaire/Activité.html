4. Visualisation des performances des deux modèles¶

L'exactitude (accuracy) est le nombre de pr√©dictions correctes divis√© par le nombre de pr√©dictions. Si l'on se fonde sur cette m√©trique, le mod√®le choisi est la SVM multi-classes, car son score est sup√©rieur √† celui du knn.

On tente de visualiser ici un peu plus pr√©cis√©ment les performances de chacun des deux mod√®les via une autre m√©trique: l'aire sous la courbe ROC.

Le souci est que l'aire sous la courbe ROC convient plut√¥t aux classifieurs mono-classe. On consid√®re donc, pour chaque classe du dataset, le probl√®me mono-classe "l'observation apartient-elle √† l'esp√®ce choisie, ou non?", et on √©value cette m√©trique pour ce probl√®me. On en fait ensuite une moyenne sur toutes les classes.

def get_auc_roc(clf, y_score, model):
     
    # Calcul de l'aire sous la courbe ROC pour chaque classe
    fpr = dict()
    tpr = dict()
    roc_auc = dict()
    thresholds = dict()
    classes = set(y_test)

    for i in classes:        
        
        # Aire sous la courbe ROC pour le probl√®me:
        # "L'observation apartient-elle √† l'esp√®ce i ?""
        y_test_ = y_test == i
        y_test_ = y_test_.astype(int)
        fpr[i], tpr[i], thresholds[i] = roc_curve(y_test_, y_score[:, i], drop_intermediate=False)
        roc_auc[i] = auc(fpr[i], tpr[i])
    
    # On r√©cup√®re la classe pour laquelle l'algorithme est le moins performant
    mini = min(roc_auc, key=roc_auc.get)
    
    # On fait la moyenne des scores obtenus
    roc_mean_ = np.mean(np.array(list(roc_auc.values())))
    
    result_ = (roc_mean_, fpr[mini], tpr[mini], roc_auc[mini], mini, model)
    return result_

def visualization(result_):
    
    fpr = result_[1]
    tpr = result_[2]
    auc = result_[3]
    mini = result_[4]
    model = result_[5]
    
    specie = labelencoder.inverse_transform([mini])[0]
    
    worst = '''Pour l'algorithme {0}:
    - La pire pr√©diction est obtenue pour l'esp√®ce {1}. 
    - La courbe ROC correspondante est obtenue avec {2} paliers.'''
    
    print(worst.format(model, specie, fpr.shape[0]))
    
    plt.figure()
    plt.plot(fpr, tpr, 'go-', label='ROC curve (area = %0.10f)' % auc)
    plt.plot([0, 1], [0, 1], 'k--')
    plt.xlim([0.0, 1.0])
    plt.ylim([0.0, 1.05])
    plt.xlabel('False Positive Rate')
    plt.ylabel('True Positive Rate')
    title = '{0} - {1} - ROC for species {2}'.format(mini, model, specie)
    plt.title(title)
    plt.legend(loc="lower right")
    plt.show()

visualization(result_knn)
